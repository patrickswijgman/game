# Game

Trick taking game.

# Ideas, rules and mechanics

- highest card wins, loser takes damage
- buff/debuff cards
- elemental mechanic? (Pokemon like)
- card effects
  - stun?
- receive damage when deck is empty?
  - prevent spamming cards
- winner of a trick takes the lead?

# Phases

- prepare enemy
  - setup character sheet
- prepare decks
  - shuffle
- enemy chooses card (only one value card, unless special effect)
- player chooses card (only one value card, unless special effect)
- reveal
- repeat until turn skipped or no more cards in hand
